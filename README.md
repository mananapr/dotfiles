## Setup
- Operating System: `Artix Linux`
- Application Launcher: `dmenu`
- Shell: `bash`
- Terminal Emulator: `kitty`
- Text Editor: `nvim`
- Video Player/Music Player/Image Viewer: `mpv`
- Web Browser: `firefox`
- Window Manager: `dwm`
- Lock Screen: `betterlockscreen`
