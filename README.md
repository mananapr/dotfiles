## Setup
- Operating System: `KISS Linux`
- Application Launcher: `dmenu`
- Shell: `bash`
- Terminal Emulator: `st`
- Text Editor: `vim`
- Video Player/Music Player/Image Viewer: `mpv`
- Web Browser: `firefox`
- Window Manager: `openbox`
- Lock Screen: `slock`
